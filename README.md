# mean-app-demo

> A simple blog developed by using MEAN (MongoDB, Express.js, Angular.js, Node.js) stack

## Usage

### Setup

```
$ npm install
```

### Developing

```
$ npm run dev
```

### Running

```
$ npm run build
$ npm start
```

### Testing

```
$ npm test
```
